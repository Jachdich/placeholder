### You need to install:
1. Flask (pip install flask)
2. Flask Forms (pip install flask-wtf)
3. SQLAlchemy (pip install flask-sqlalchemy)
4. Bcrypt (pip install bcrypt)
5. Flask Login (pip install flask-login)
